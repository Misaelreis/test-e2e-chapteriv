## __Requisitos__
1. [Node](https://nodejs.org/pt-br/)
2. Chrome

## __Instalando Dependências__
1. Rodar o comando 
```
npm install
```
2. Para abrir o Cypress Application use o comando
```
npm run cy:open
````

3. Para rodar o Cypress Headless use o comando
```
npm run cy:run
````
